# Pixel Car Racing

A pixel art car racing game for Android (web-based)

## Features
- Pixel art graphics
- Multiple car colors to choose from
- Increasing difficulty as you progress
- Touch controls (tap left/right to steer)
- High score tracking
- Portrait mode optimized

## How to Play
- Tap the left side of the screen to move left
- Tap the right side of the screen to move right
- Avoid enemy cars
- Score increases as you drive faster

## Files
- `pixel-car-racing.html` - Standalone HTML game (open in browser)
- `www/` - Web app source files
- `dist/` - PWA files (can be converted to APK)

## Convert to APK
Use online services:
1. Go to https://webintoapp.com
2. Upload `pixel-car-racing.html`
3. Download APK

Or use PWA Builder: https://www.pwabuilder.com

## Installation
1. Download the HTML file
2. Open in mobile browser
3. Add to home screen (PWA)
4. Or convert to APK using above method
